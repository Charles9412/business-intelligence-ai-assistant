# Portfolio Summary

## Project Description

Business Intelligence AI Assistant is a local-first RAG + SQL demo that answers business questions over synthetic payment operations data. It routes questions to document retrieval, safe SQLite analytics, or a hybrid workflow that combines documented business rules with database calculations. The project demonstrates grounded answer generation, SQL safety controls, synthetic data design, and a portfolio-ready Gradio interface.

## Key Skills Demonstrated

- Local LLM integration with an OpenAI-compatible API.
- Retrieval-Augmented Generation over markdown business documents.
- Persistent vector search with ChromaDB and sentence-transformers.
- Synthetic data generation for realistic BI scenarios.
- SQLite schema design and analytical querying.
- Safe SQL generation with read-only validation.
- Deterministic question routing across RAG, SQL, and Hybrid workflows.
- Prompt design for grounded answers and SQL generation.
- Gradio UI development for explainable assistant traces.
- Unit testing with mocked LLM dependencies.
- Documentation and portfolio storytelling.
